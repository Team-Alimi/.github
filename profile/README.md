# 📢 IN:FORM

> **인하대학교 학내 행사와 동아리 소식을 한곳에 모은 통합 정보 플랫폼**

IN:FORM은 흩어져 있는 학내 행사·동아리 정보를  
**캘린더 기반으로 직관적으로 제공**하여  
학생들이 놓치지 않고 참여할 수 있도록 돕는 서비스입니다


## 🚀 What We Build
- 학내 행사 및 동아리 정보를 **캘린더 중심 UI**로 제공하는 웹 서비스
- 학생 입장에서 **한눈에 보기 쉽고, 찾기 쉬운 정보 구조**를 지향합니다

### ✨ 주요 기능
- 📅 **월간 캘린더**로 모든 행사 일정 한눈에 확인
- 🎪 **행사 목록 조회** 및 상세 정보 제공
- 🏛️ **동아리 정보 페이지** 및 상세 소개
- 🔍 **동아리 · 행사 통합 검색**
- 📱 다양한 디바이스를 고려한 **반응형 디자인**
